Resonance of Fate, available for the Playstation 3 and Xbox 360 (I purchased for the former), has a vast amount of data associated with it, as RPGs usually do.

Because I usually do, I started keeping track of levels and etcetera for the game, which is available as an XML data file.

The data is currently raw (and as of this post need to finish adding data that I've got on paper and on another computer), without styling, but I've been sitting on the information for far too long.
